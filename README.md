# Roll20 Spell Manager 🔮

Uma aplicação web desenvolvida em React, TypeScript e Vite para gerenciar e preparar listas de magias e exportá-las diretamente para fichas clássicas **D&D 5e (2014/OGL)** do Roll20 em conjunto com a extensão **Enhanced VTT**.

## 🚀 Principais Recursos
- **Grimório Dinâmico**: Visualize, adicione, remova e edite magias com filtros avançados de nível, escola de magia, concentração, ritual e status de preparação.
- **Preparação de Magias**: Interface dedicada para marcar magias preparadas e sempre preparadas do seu personagem.
- **Edição sem Perdas**: Importa arquivos JSON exportados pelo Roll20, modifica as magias e gera o novo arquivo preservando todos os atributos originais intactos.
- **Visualização Premium**: Tema escuro com layout responsivo e moderno.

---

## 💻 Integração com a Extensão (Como Usar)
1. Exporte a ficha do seu personagem no Roll20 utilizando o painel da extensão **Enhanced VTT**.
2. Abra esta ferramenta web e carregue/arraste o arquivo `.json` exportado.
3. Altere suas magias (adicione novos truques ou magias de círculos superiores, prepare sua lista diária).
4. Clique em **"Export Roll20 Character JSON"** para baixar a versão atualizada da ficha.
5. Volte para o painel da extensão no Roll20, clique em **"Importar"** e selecione o arquivo editado para carregar as modificações automaticamente na ficha sem causar duplicados.
